### 目录结构
    - assets (静态资源)
        - javascripts
        - stylesheet  
        - libs
        - imgs
    - components
        radar (组件的目录,每个组件有个html文件用于演示组件如何使用)
        ...
    - data (存放数据的目录)
    - views (大屏模板)

### 组件的定义
    - 组件（Component）是对数据和方法的简单封装。简而言之，组件就是对象。
    - 组件一个重要的特点是可定制性，目的是为了以后的可以复用，减少重复的劳动。（可定制性包括数据和样式等）
